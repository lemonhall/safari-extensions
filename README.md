safari-extensions
=================

自己开发的safari扩展，safari extension for myself

doban-clear
=================
将豆瓣的右边栏清空，加上自己的日记标签，以及我的广播的小扩展，主要就是演示了注入式script的使用

douban-share
=================
safari的bar button，在工具栏上的扩展，使用了global与注入script的通讯

digg-share-douban
=================
digg-share则是给digg-reader加上分享到豆瓣能力的小脚本，只使用注入式script即可
